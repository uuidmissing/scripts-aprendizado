HTML Avançado: Divs, Classes e Estrutura

Este documento explica conceitos um pouco mais avançados de HTML, como <div>, classes e organização semântica.

O que é HTML?

HTML significa HyperText Markup Language. Ele define a estrutura de uma página Web usando elementos como:

  • <h1>, <h2> — títulos e subtítulos
  • <p> — parágrafos de texto
  • <a> — links
  • <img> — imagens

O que é uma <div>?

<div> é um contêiner genérico usado para agrupar conteúdo quando não há um elemento semântico específico apropriado.

Use <div> para estruturar o layout, especialmente quando precisar aplicar estilos ou scripts.

<div class="card"> <h2>Título</h2> <p>Conteúdo do cartão.</p> </div>

O que são classes?

Classes são atributos atribuídos a elementos HTML para aplicar estilos ou selecionar elementos via CSS/JavaScript.

<div class="card destaque"> <h3>Cartão de Exemplo</h3> <p>Use classes para estilizar ou controlar comportamento.</p> </div>

Um elemento pode ter várias classes separadas por espaço:

<div class="card destaque grande"> ... </div>

Exemplo prático com <div> e classes

Veja abaixo um layout simples usando divs e classes:

<div class="container"> <div class="header"> <h1>Cabeçalho</h1> </div> <div class="content"> <div class="card"> <h2>Cartão 1</h2> <p>Texto dentro do cartão.</p> </div> <div class="card"> <h2>Cartão 2</h2> <p>Outro texto dentro de outro cartão.</p> </div> </div> </div>

Esse código cria uma estrutura clara para aplicar CSS e JavaScript depois.

Semântica vs. estrutura

Embora <div> seja útil, use elementos semânticos sempre que possível:

  • <header> para cabeçalho
  • <nav> para navegação
  • <article> para conteúdo independente
  • <section> para seções temáticas
  • <footer> para rodapé

Use <div> quando não houver elemento semântico adequado, mas prefira HTML semântico para melhor acessibilidade.

Dica

Classes são especialmente úteis quando combinadas com CSS moderno e frameworks como Tailwind, Bootstrap ou outras bibliotecas.
