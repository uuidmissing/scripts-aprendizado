Tags HTML Mais Usadas

Conheça as principais tags HTML e suas funcionalidades:

<h1> a <h6>

Títulos e subtítulos. <h1> é o mais importante, <h6> o menos.

<p>

Parágrafo de texto. Use para blocos de conteúdo textual.

<a>

Link. Exemplo: <a href="url">clique aqui</a>

<img>

Imagem. Exemplo: <img src="imagem.jpg" alt="descrição">

<div>

Contêiner genérico para agrupar elementos e aplicar estilos.

<span>

Contêiner inline. Menor que <div>, para textos.

<ul>, <ol>, <li>

<ul> lista sem ordem, <ol> com ordem. <li> é cada item.

<table>, <tr>, <td>, <th>

Tabelas. Use para dados tabulares, não para layout.

<form>

Formulário para coleta de dados. Contém <input>, <textarea>, etc.

<input>

Entrada de dados. Tipos: text, email, password, number, checkbox, radio, etc.

<button>

Botão clicável. Pode estar dentro de <form> ou ser usado com JavaScript.

<select>, <option>

Menu suspenso. <option> são as opções disponíveis.

<label>

Rótulo para inputs. Melhora acessibilidade e usabilidade.

<section>, <article>, <nav>, <footer>, <header>

Tags semânticas para melhor estrutura e acessibilidade.

<br>, <hr>

<br> quebra linha. <hr> linha horizontal.

Exemplo de estrutura básica:

<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <title>Minha Página</title> <link rel="stylesheet" href="estilo.css"> </head> <body> <header> <h1>Bem-vindo</h1> <nav> <a href="#">Home</a> <a href="#">Sobre</a> </nav> </header> <main> <article> <h2>Artigo</h2> <p>Conteúdo aqui...</p> </article> </main> <footer> <p>© 2026</p> </footer> <script src="script.js"></script> </body> </html>

CSS Prático e Exemplos

Cores e Gradientes

CSS permite cores sólidas, gradientes e muito mais:

/* Cor sólida */ background: #667eea; /* Cor RGB */ background: rgb(102, 126, 234); /* Gradiente linear */ background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); /* Gradiente radial */ background: radial-gradient(circle, #667eea 0%, #764ba2 100%);

Flexbox - Layout flexível

Flexbox ajuda a criar layouts responsivos:

.container { display: flex; justify-content: center; /* Alinha horizontalmente */ align-items: center; /* Alinha verticalmente */ gap: 10px; /* Espaço entre itens */ flex-wrap: wrap; /* Quebra em múltiplas linhas */ } .item { flex: 1; /* Crescem igualmente */ min-width: 200px; }

Transições e Animações

Adicione movimento e suavidade com CSS:

/* Transição suave */ button { transition: all 0.3s ease; } /* Animação */ @keyframes slide { from { transform: translateX(-100px); } to { transform: translateX(0); } } .elemento { animation: slide 0.5s ease; }

Sombras e Bordas

/* Sombra de caixa */ box-shadow: 0 10px 30px rgba(102, 126, 234, 0.3); /* Borda arredondada */ border-radius: 8px; /* Sombra de texto */ text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2);

Diferença entre Grid e Flexbox

Característica Grid Flexbox
Dimensões 2D (linhas e colunas) 1D (linha ou coluna)
Melhor para Layouts complexos Alinhamento e distribuição
Uso comum Página inteira Barra de navegação, cards
✓ Dica: Use Grid para layouts gerais e Flexbox para alinhamento de componentes.
