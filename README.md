# comandos_para_aprendizado

Os scripts neste repositório são para estudo e revisão, ajudando no aprendizado de Bash, Python e Git.

## Novos tutoriais

- `TUTORIAL_VSCODE_BASICO.md`: tutorial básico para começar a usar o VS Code.
- `git/GIT_UNIFICADO.md`: guia unificado de Git com comandos e práticas recomendadas.
- `html/arquivo_teste.html`: exemplo HTML limpo e moderno.
- `html/explicacao_html_avancado.html`: explicações sobre HTML, `div`, `class` e conceitos um pouco mais avançados.
- `html/html_css_javascript_completo.html`: **Guia completo e interativo** com tags HTML, CSS prático, JavaScript com exemplos, formulários e layouts (Grid/Flexbox).
